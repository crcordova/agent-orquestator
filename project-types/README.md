# Project Types

Domain-specific patterns and checklists for common project types. The planner reads the relevant guide when classifying a project.

| Guide | Use When |
|-------|----------|
| [SaaS Application](saas.md) | Multi-tenant web app with auth, billing, dashboard |
| [Microservice](microservice.md) | Independent service with its own data and API |
| [ML Project](ml-project.md) | Data pipeline, model training, inference service |
| [REST API](rest-api.md) | HTTP API with CRUD operations |
| [GraphQL API](graphql-api.md) | GraphQL schema with resolvers |
| [Mobile App](mobile-app.md) | iOS/Android/cross-platform app |
| [CLI Tool](cli-tool.md) | Command-line application |
| [Monorepo](monorepo.md) | Multiple packages in one repository |
